## Pulse 001 — Phase 1: Public Spine + Design System + ADRs

**Date:** 2026-06-27
**Type:** Probe pulse — live public repo as proof-of-work
**Repo:** https://github.com/davidnunez/the-bench

### What this pulse contains

The Phase-1 pulse IS the live public repo itself (D-22). It ships:

- **Spine:** [`README.md`](README.md) — before→alive thesis, environment-as-provenance, the over-engineering-as-hook angle, Signal → Probe → Pulse framing, Board back-link to davidnunez.com
- **License:** [`LICENSE`](LICENSE) — MIT 2026 David Nunez, whole repo, maximally open
- **Design system:** [`design-system/tokens.yaml`](design-system/tokens.yaml) — full semantic brand system: 8 Signal Path types (signal/probe/pulse/broadcast/module/cadence/component/schematic) + base + roles + reserved/unbound inks + typography + provenance. Pinned snapshot of the Board's color semantics
- **First generated label:** [`design-system/labels/the-bench-labels.svg`](design-system/labels/the-bench-labels.svg) — `the-bench` wordmark (aubergine on cream, probe identity) + eight-label ink-legend card (each type as a swatch with meaning, IBM Plex Mono). Generated by `scripts/generate-labels.py` from `tokens.yaml`; `make labels` is the build target
- **Four load-bearing ADRs:**
  - [`decisions/ADR-001-openscad-include-vs-use.md`](decisions/ADR-001-openscad-include-vs-use.md) — `include` vs `use` + `$fn` convention; locks the silent-`undef` failure mode out before any `.scad` file is written
  - [`decisions/ADR-002-dollhouse-print-scale.md`](decisions/ADR-002-dollhouse-print-scale.md) — 1:25 scale locked; Bambu A1 256mm bed fits the full footprint
  - [`decisions/ADR-003-modular-dollhouse-walls.md`](decisions/ADR-003-modular-dollhouse-walls.md) — floor base + swappable walls via printed joinery; joint mechanism deferred to Phase 6
  - [`decisions/ADR-004-semantic-color-integrity.md`](decisions/ADR-004-semantic-color-integrity.md) — brand inks carry Signal Path type meaning always; reflect-don't-invent; Board boundary

### Signal Path framing

The Bench is a **Probe** (hands-on exploration, aubergine ink).
This pulse is a **Pulse** (ephemeral share of the phase's proof-of-work, soft coral by type — though emitted in writing here, not ink).
A stranger landing on the repo should see a real human turning his environment into a reproducible, expressive, lovingly over-engineered system — and want to follow along.
